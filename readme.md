## Instrucciones para funcionamiento del sistema

- Tener PHP >= 7.1.3
- Realizar git clone https://github.com/GerarPacheco/cts.git
- Crear Base de Datos con nombre cts
- Editar archivo .env (si es necesario)
- Ejecutar composer install
- Ejecutar php artisan migrate --seed
- Iniciar aplicación
